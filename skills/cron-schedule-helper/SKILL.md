# Cron Schedule Helper Skill

將自然語言轉換為 cron 表達式，並提供執行時間預覽。

## 核心功能

1. **自然語言轉 Cron** - 將描述轉換為 cron 表達式
2. **Cron 語法驗證** - 檢查 cron 表達式正確性
3. **執行時間預覽** - 顯示接下來的執行時間
4. **格式說明** - 解釋 cron 表達式的意義
5. **常見模式庫** - 提供常用 cron 範本

## 使用時機

當使用者提出以下需求時，啟用此 Skill：

- "每天凌晨2點執行"
- "每30分鐘跑一次"
- "這個 cron 表達式是什麼意思"
- "0 2 * * * 下次什麼時候執行"
- "週一到週五早上9點"
- "設定排程時間"

## Cron 格式說明

### Argo Workflow 格式（5 位數）

```
 ┌─── 分鐘 (0 - 59)
 │  ┌─── 小時 (0 - 23)
 │  │  ┌─── 日 (1 - 31)
 │  │  │  ┌─── 月 (1 - 12)
 │  │  │  │  ┌─── 週 (0 - 7, 0和7都是週日)
 │  │  │  │  │
 *  *  *  *  *
```

**重要**: Argo Workflow 使用 5 位數格式，**沒有秒位**，最小單位是**分鐘**。

### 特殊字元

| 字元 | 意義 | 範例 |
|------|------|------|
| `*` | 任意值 | `* * * * *` = 每分鐘 |
| `,` | 列舉 | `0,30 * * * *` = 每小時的0分和30分 |
| `-` | 範圍 | `0 9-17 * * *` = 9:00到17:00每小時 |
| `/` | 間隔 | `*/15 * * * *` = 每15分鐘 |

### 錯誤格式（6 位數包含秒）

```
❌ 0 0 2 * * *  (6 位數 - 包含秒位)
✅ 0 2 * * *    (5 位數 - Argo Workflow 正確格式)
```

## 自然語言轉換

### 基本模式

#### 每天執行

| 描述 | Cron 表達式 | 說明 |
|------|------------|------|
| "每天凌晨2點" | `0 2 * * *` | 每天 02:00 執行 |
| "每天早上9點" | `0 9 * * *` | 每天 09:00 執行 |
| "每天中午12點" | `0 12 * * *` | 每天 12:00 執行 |
| "每天下午3點" | `0 15 * * *` | 每天 15:00 執行 |
| "每天晚上8點" | `0 20 * * *` | 每天 20:00 執行 |
| "每天半夜12點" | `0 0 * * *` | 每天 00:00 執行（midnight） |

#### 間隔執行

| 描述 | Cron 表達式 | 說明 |
|------|------------|------|
| "每5分鐘" | `*/5 * * * *` | 每5分鐘一次 |
| "每10分鐘" | `*/10 * * * *` | 每10分鐘一次 |
| "每15分鐘" | `*/15 * * * *` | 每15分鐘一次 |
| "每30分鐘" | `*/30 * * * *` | 每30分鐘一次 |
| "每小時" | `0 * * * *` | 每小時整點 |
| "每2小時" | `0 */2 * * *` | 每2小時整點 |
| "每4小時" | `0 */4 * * *` | 每4小時整點 |

#### 工作日執行

| 描述 | Cron 表達式 | 說明 |
|------|------------|------|
| "週一到週五早上9點" | `0 9 * * 1-5` | 平日 09:00 |
| "週一到週五下午6點" | `0 18 * * 1-5` | 平日 18:00 |
| "週末早上10點" | `0 10 * * 0,6` | 週六日 10:00 |
| "週一早上8點" | `0 8 * * 1` | 每週一 08:00 |
| "週五下午5點" | `0 17 * * 5` | 每週五 17:00 |

#### 每月執行

| 描述 | Cron 表達式 | 說明 |
|------|------------|------|
| "每月1號凌晨2點" | `0 2 1 * *` | 每月第1天 02:00 |
| "每月15號中午" | `0 12 15 * *` | 每月第15天 12:00 |
| "每月最後一天" | `0 0 28-31 * *` | (需配合月份判斷) |
| "每月第一個週一" | `0 9 1-7 * 1` | 每月1-7號的週一 |

#### 營業時間執行

| 描述 | Cron 表達式 | 說明 |
|------|------------|------|
| "營業時間每小時 (9-17點)" | `0 9-17 * * *` | 09:00到17:00每小時 |
| "營業時間每15分鐘" | `*/15 9-17 * * *` | 09:00到17:00每15分鐘 |
| "營業日早上9點 (週一到週五)" | `0 9 * * 1-5` | 平日 09:00 |
| "營業時間開始和結束" | `0 9,18 * * 1-5` | 平日 09:00 和 18:00 |

### 複雜模式

#### 多時間點

```
"每天早上9點、中午12點、下午6點"
→ 0 9,12,18 * * *
```

#### 時間範圍內間隔

```
"早上9點到下午5點，每30分鐘"
→ */30 9-17 * * *
```

#### 特定日期組合

```
"週一和週三早上10點"
→ 0 10 * * 1,3
```

#### 工作日特定時間

```
"週一到週五，早上9點和下午2點"
→ 0 9,14 * * 1-5
```

### 步驟 2: 組合 Cron 表達式

**組合規則**:
1. 確定分鐘位（預設 0，除非有間隔）
2. 確定小時位（根據時間描述）
3. 確定日期位（預設 *，除非指定特定日期）
4. 確定月份位（預設 *，除非指定特定月份）
5. 確定星期位（根據星期描述）

### 步驟 3: 驗證表達式

**驗證項目**:
- ✅ 5 位數格式（不是 6 位數）

- ✅ 特殊字元使用正確
- ✅ 邏輯合理（如不會同時指定日期和星期）

## 驗證功能

### 語法驗證

**檢查項目**:
1. **位數檢查**
   ```
   ✅ "0 2 * * *"  (5 位數)
   ❌ "0 0 2 * * *"  (6 位數 - 包含秒)
   ❌ "0 2 * *"  (4 位數 - 缺少星期)
   ```

2. **數值範圍**
   ```
   分鐘: 0-59  ✅ "30 * * * *"  ❌ "60 * * * *"
   小時: 0-23  ✅ "0 23 * * *"  ❌ "0 24 * * *"
   日:   1-31  ✅ "0 0 15 * *"  ❌ "0 0 32 * *"
   月:   1-12  ✅ "0 0 1 12 *"  ❌ "0 0 1 13 *"
   週:   0-7   ✅ "0 0 * * 0"   ❌ "0 0 * * 8"
   ```

3. **特殊字元**
   ```
   ✅ "*/15 * * * *"  (間隔)
   ✅ "0 9-17 * * *"  (範圍)
   ✅ "0 9,12,18 * * *"  (列舉)
   ❌ "0 9--17 * * *"  (雙連字號)
   ❌ "0 9, * * *"  (逗號後沒有值)
   ```

### 邏輯驗證

**衝突檢測**:
```
❌ 警告: "0 2 15 * 1"
   同時指定「每月15號」和「週一」可能導致預期外的行為
   建議: 擇一使用

❌ 錯誤: "0 2 31 2 *"
   2月沒有31號，此排程永遠不會執行

✅ 安全: "0 2 * * 1-5"
   平日執行，邏輯清晰
```

- 位數: 5 ✅ (正確，Argo Workflow 格式)

## 常見模式庫

### 日常任務

```yaml
# 每日報表 (凌晨2點)
schedule: "0 2 * * *"

# 每日備份 (凌晨3點)
schedule: "0 3 * * *"

# 每日資料同步 (每小時)
schedule: "0 * * * *"

# 每日清理 (半夜12點)
schedule: "0 0 * * *"
```

### 批次處理

```yaml
# 小批次 (每5分鐘)
schedule: "*/5 * * * *"

# 中批次 (每15分鐘)
schedule: "*/15 * * * *"

# 大批次 (每小時)
schedule: "0 * * * *"
```

### 業務時間

```yaml
# 營業開始 (早上9點)
schedule: "0 9 * * 1-5"

# 營業結束 (下午6點)
schedule: "0 18 * * 1-5"

# 營業時間每小時 (9-17點)
schedule: "0 9-17 * * 1-5"

# 午休前 (中午12點)
schedule: "0 12 * * 1-5"
```

### 週期任務

```yaml
# 每週一早上 (週報)
schedule: "0 9 * * 1"

# 每月1號 (月報)
schedule: "0 2 1 * *"

# 每季初 (1,4,7,10月1號)
schedule: "0 2 1 1,4,7,10 *"

# 週末維護 (週日凌晨2點)
schedule: "0 2 * * 0"
```
